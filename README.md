# Shapeless bot
A MsgRoom bot for [Windows96.net](https://windows96.net). The bot joins as `=>|<=`, if parameter `name_lasts` is not `true` and `default_name` is `=>|<=`.

The full code is not shared, but some of its parts are published here.
